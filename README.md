# Новый сайт кактус клуба #


В связи с тем что текущий сайт кактус клуба устарел по своему оформлению предлагаю его переделать. Новый сайт должен соотвествовать следующим требованиям:
1. Современный дизай
2. Авторизация для членов клуба
3. Удобная поддержка
4. Расширенный контент
	* Все что есть на текущем сайте, должно быть и на новом
	* Фото и описания видов и растений
	* Меропрриятия клуба !!!!
	* История клуба 
	* Члены клуба
	* Интеграция с соц. сетями и возможность делать репосты мероприятий клуба
Пожелание:
Добавить геолокацию собитий и место произрастания растений. 


В качаестве ЯП будет использовани язык Python, в качастве БД Postgresql
